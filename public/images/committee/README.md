# Foto Kepanitiaan

Silakan masukkan (copy-paste) semua foto-foto anggota panitia ke dalam folder ini.

**Tips Penamaan:**
Agar nanti mudah dipanggil di dalam kode program, sangat disarankan untuk mengubah nama file fotonya menggunakan sesuatu yang unik namun konsisten.
Contoh: menggunakan NPM (`240712848.jpg`, `240327977.jpg`) atau nama panggilan singkat (`roy.jpg`, `stevany.png`).
